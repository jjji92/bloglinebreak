# 네이버 블로그 글 자동 줄바꿈 & 가독성 정리 도구

네이버 블로그 글 작성 시 가독성을 높이기 위해  
문단 간 줄바꿈을 자동으로 정리해주는 웹 기반 도구입니다.

본 프로젝트의 목적은 **검색 유입 기반 트래픽 확보 + 광고 수익화**입니다.

---

## 1. 프로젝트 목표

- 네이버 블로그 작성자의 반복적인 불편(줄바꿈, 문단 정리) 해결
- 검색 유입 중심의 단일 기능 웹사이트 구축
- 애드센스 기반 광고 수익 실험
- MVP → 수익성 검증 → 기능 확장

---

## 2. 타겟 사용자

- 네이버 블로그 초보 사용자
- 체험단 / 리뷰 작성자
- 블로그 글 가독성에 민감하지만 정리 시간이 부족한 사용자

---

## 3. 핵심 기능 (MVP)

### 3.1 텍스트 자동 줄바꿈
- 사용자가 텍스트를 붙여넣으면
- 문장 단위 또는 문단 단위로 자동 줄바꿈
- 불필요한 연속 공백 제거

### 3.2 네이버 블로그 최적 줄 간격
- 문단 사이 줄바꿈 횟수 고정 (예: 2줄)
- 모바일 기준 가독성 고려

### 3.3 결과 미리보기
- 변환 전 / 변환 후 텍스트 비교
- 복사 버튼 제공

---

## 4. 수익 구조

### 4.1 1차 수익 (필수)
- Google AdSense
  - 상단 배너
  - 결과 영역 하단 광고

### 4.2 2차 수익 (확장 가능)
- 블로그 관련 제휴 링크
  - 블로그 강의
  - 마케팅 툴
- (추후) 프리미엄 기능
  - 글자 수 제한 해제
  - 저장 기능
  - 맞춤 문단 스타일

---

## 5. SEO 전략

### 5.1 주요 키워드
- 네이버 블로그 줄바꿈
- 블로그 가독성
- 블로그 글 정리
- 네이버 블로그 글쓰기 팁

### 5.2 검색 유입용 콘텐츠
- 네이버 블로그 줄바꿈 기준 설명 페이지
- 가독성 좋은 글 예시 비교 페이지
- 블로그 체류시간 늘리는 방법 정리

> 도구 페이지 + 정보 페이지 병행 구성

---

## 6. 페이지 구성

- `/` : 줄바꿈 도구 메인
- `/guide` : 네이버 블로그 가독성 가이드
- `/examples` : 전/후 비교 예시
- `/privacy` : 개인정보 처리방침 (애드센스 필수)
- `/terms` : 이용약관
- `/about` : 서비스 소개
- `/contact` : 문의하기
- `/faq` : 자주 묻는 질문(FAQ)

---

## 7. 기술 스택

- Frontend: React + Vite
- Styling: Tailwind CSS
- Deployment: Vercel
- Ads: Google AdSense

> 서버리스, 로그인 없음, 데이터 저장 없음

---

## 7.1 실행 방법 (pnpm)

```bash
pnpm install
pnpm run build
pnpm run preview
```

> 주의: 개발 서버 실행(`pnpm run dev`)은 로컬 확인 용도로만 사용하세요.

---

## 7.2 배포 환경 변수(권장)

에드센스 심사/신뢰도 관점에서 **문의처/운영자 정보**가 중요합니다.

- `VITE_CONTACT_EMAIL`: 문의 이메일 (예: `help@yourdomain.com`)
- `VITE_SITE_OWNER`: 운영자/팀 이름 (선택)
- `VITE_SITE_NAME`: 사이트 이름 (선택)
- `VITE_ENABLE_ADS`: 광고 활성화(승인 후에만 설정 권장)

> 위 기준 충족 시 기능 확장 고려
